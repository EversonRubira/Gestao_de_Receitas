# Sistema de Gestão de Receitas

Projeto PIS 2025/2026 - CTeSP TPSI

## Tecnologias

- Node.js + Express
- MySQL
- EJS
- API REST

## Como usar

1. Instalar dependências:
```bash
npm install
```

2. Criar a base de dados:
```bash
mysql -u root -p < database/schema.sql
```

3. Iniciar servidor:
```bash
npm start
```

## Funcionalidades

- CRUD de receitas
- Login/registo
- Backoffice (admin)
- Frontoffice (público)
- API REST
- Integração TheMealDB
